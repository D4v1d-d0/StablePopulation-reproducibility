# StablePopulation reproducibility files for the ECOSISTEMAS ecoinformatics note

This repository contains the input data, external R script, output files, and final figure formats required to reproduce the comparative Weibull fits presented in the ECOSISTEMAS ecoinformatics note on **StablePopulation**.

The repository reproduces the three example cases used in the manuscript:

- *Castor canadensis* (best fit)
- *Ovis dalli* (intermediate fit)
- *Rangifer tarandus* (poorest fit)

## Repository contents

- `Inputs_3_examples.xlsx`  
  Input file used by the external reproducibility script.

- `make_figure_and_outputs.R`  
  External R script used to generate the outputs and Figure 1.

- `Outputs_3_examples.xlsx`  
  Output file generated by the script, including the fitted results.

- `Figure1_WeibullExamples.pdf`  
  Final Figure 1 in PDF format.

- `Figure1_WeibullExamples_300dpi.png`  
  Final Figure 1 in PNG format (300 dpi).

- `Figure1_WeibullExamples_300dpi.jpg`  
  Final Figure 1 in JPG format (300 dpi).

- `Figure1_WeibullExamples_300dpi.tiff`  
  Final Figure 1 in TIFF format (300 dpi).

## Purpose of this repository

The `StablePopulation` R package performs the core demographic calculations. However, the selection of the final pair \((\alpha^*, \beta^*)\) for the three manuscript examples was carried out **outside the package** using an external R script.

This repository provides the materials needed to reproduce that external analysis step, including:

1. reading the prepared input file,
2. running the beta sweep,
3. computing the corresponding alpha value for each beta,
4. evaluating fit quality,
5. generating the output workbook,
6. exporting Figure 1 in multiple formats.

## Requirements

The script requires R and the following packages:

- `readxl`
- `openxlsx`
- `StablePopulation`

If needed, install them in R with:

```r
install.packages(c("readxl", "openxlsx", "StablePopulation"))
```

## How to reproduce the analysis

1. Download or clone this repository.
2. Open R in the repository folder.
3. Ensure that `Inputs_3_examples.xlsx` and `make_figure_and_outputs.R` are in the working directory.
4. Run:

```r
source("make_figure_and_outputs.R")
```

## Expected outputs

Running the script should generate:

- `Outputs_3_examples.xlsx`
- `Figure1_WeibullExamples.pdf`
- `Figure1_WeibullExamples_300dpi.png`
- `Figure1_WeibullExamples_300dpi.jpg`
- `Figure1_WeibullExamples_300dpi.tiff`

## Relation to the StablePopulation package

This repository is complementary to the CRAN package:

Palacios-Morales, D., Rodríguez-Gómez, G., & Martín-González, J. A. (2025). *StablePopulation: Calculates Alpha for a Stable Population* (R package version 1.0.3). CRAN. https://doi.org/10.32614/CRAN.package.StablePopulation

The package performs the core demographic calculations, while this repository contains the external reproducibility materials used for the manuscript-specific comparison and figure generation.
