# Reproducibility materials for the ECOSISTEMAS ecoinformatics note on StablePopulation

This repository contains the input data, external R script, output files, and final figure formats required to reproduce the comparative Weibull fits presented in the ECOSISTEMAS ecoinformatics note on **StablePopulation**.

The original analysis described in the note was carried out with **StablePopulation 1.0.3**, the CRAN release documented in the manuscript. The reproducibility script relies only on the long-standing core functions `find_alphas()` and `calculate_population()`. It therefore does not depend on later development functions and will continue to work with later compatible versions of the package.

The current reproducibility workflow includes three example cases:

- *Castor canadensis*
- *Ovis dalli*
- *Rupicapra rupicapra*

## Start here

For a complete explanation of the practical case, from the input data to the selected Weibull profiles and Figure 1, read the [step-by-step practical workflow](docs/practical_case_workflow.md).

## Repository contents

- `Inputs_3_examples.xlsx`  
  Input workbook used by the manuscript-specific reproducibility script.
  
- `make_figure_and_outputs.R`

- `Outputs_3_examples.xlsx`

- `Figure1_WeibullExamples.pdf`

- `Figure1_WeibullExamples_300dpi.jpg`  
  Final Figure 1 in JPG format (300 dpi).

- `Figure1_WeibullExamples_300dpi.tiff`  
  Final Figure 1 in TIFF format (300 dpi).

- `Figure1_WeibullExamples.eps`  
  Optional EPS version of Figure 1, generated only if requested in the script.

- `docs/practical_case_workflow.md`  
  Step-by-step tutorial for the manuscript-specific practical case. It explains the input data, beta scan, alpha estimation, profile reconstruction, ECM/RMSE calculation, model selection, and Figure 1 generation.

  Example material documenting the Excel-based `run_analysis()` interface:
  - `Input_Data.xlsx` illustrates the input workbook layout expected by the function.
  - `outputs/` contains the corresponding output workbooks generated from the example worksheets.
  - `README.md` explains the scope and structure of this example.

## Two complementary workflows

This repository documents two related but distinct uses of **StablePopulation**.

### Manuscript practical case

The practical case used in the ECOSISTEMAS note starts from observed survivorship (`lx_obs`) and age-specific fertility (`mx`). The external script explores a predefined grid of `beta` values, obtains the corresponding `alpha` value compatible with `R0 = 1`, evaluates the fit to the observed profile through ECM and RMSE, and retains the best-fitting profile.

```text
Inputs_3_examples.xlsx
        ↓
make_figure_and_outputs.R
        ↓
find_alphas() + calculate_population()
        ↓
beta sweep, ECM/RMSE, selected profile and Figure 1
```

### Excel interface example

The material in `examples/run_analysis/` illustrates a separate fixed-`beta` workflow based on `run_analysis()`. It is useful for reconstructing a profile from a prepared Excel workbook, but it does not itself perform the beta sweep, comparison with observed survivorship, or Figure 1 generation used in the manuscript practical case.

```text
examples/run_analysis/Input_Data.xlsx
        ↓
run_analysis()
        ↓
fixed-beta reconstruction and Excel output
```

## Purpose of this repository

The `StablePopulation` R package performs the core demographic calculations. The manuscript-specific comparison and figure generation are carried out outside the package by `make_figure_and_outputs.R`.

This repository provides the materials needed to reproduce that external analysis step, including:

1. reading the prepared input workbook;
2. running a beta sweep;
3. calculating the corresponding alpha value for each beta;
4. evaluating fit quality through ECM and RMSE;
5. exporting the output workbook; and
6. generating Figure 1 in multiple formats.

## Requirements

The script requires R and the following packages:

- `readxl`
- `openxlsx`
- `StablePopulation`

If any of them are missing, the script installs them automatically. If `StablePopulation` is already installed, the script uses the installed version without imposing a version restriction.

If needed, the packages can also be installed manually with:

```r
install.packages(c("readxl", "openxlsx", "StablePopulation"))
```

## How to reproduce the analysis

1. Download or clone this repository.
2. Open R in the repository folder.
3. Ensure that `Inputs_3_examples.xlsx` and `make_figure_and_outputs.R` are in the working directory.
4. Run:

```r
source("make_figure_and_outputs.R")
```

## Expected outputs

Running the script should generate:

- `Outputs_3_examples.xlsx`
- `Figure1_WeibullExamples.pdf`
- `Figure1_WeibullExamples_300dpi.png`
- `Figure1_WeibullExamples_300dpi.jpg`
- `Figure1_WeibullExamples_300dpi.tiff`

## Relation to the StablePopulation package

This repository is complementary to the CRAN package:

Palacios-Morales, D., Rodríguez-Gómez, G., & Martín-González, J. A. (2025). *StablePopulation: Calculates Alpha for a Stable Population* (R package version 1.0.3). CRAN. https://doi.org/10.32614/CRAN.package.StablePopulation

The ECOSISTEMAS note documents version 1.0.3 because it was the version used for the original analysis. This repository uses only the core functions already available in that release, while remaining independent of later development functions that may be added to the package.
